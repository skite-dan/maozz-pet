2 发现的Bug清单（第一轮）

Bug ID 严重程度 模块 问题描述 状态

BUG-006 低 UI 部分CSS类未定义（18个） 已记录

3 修复措施

  1. models/index.js: 将 pool.execute() 改为 pool.query()，支持字符串拼接SQL + 参数化占位符混合使用
  2. routes/auth.js: 为 /me 路由添加 auth 中间件，导入 auth 和 optionalAuth
  3. routes/messages.js: 添加 queryOne 导入
  4. middlewares/admin.js: 改进权限验证，区分无token(401)和角色不足(403)
  5. app.js: 添加自定义404错误页面处理，区分API和页面请求
  6. views/error.ejs: 创建自定义404/500错误页面模板
